# 📊 Sales Analysis Using Pivot Tables & PivotCharts

This project demonstrates how to analyze bike sales data using **Excel Pivot Tables** and **PivotCharts**.  
It includes customer segmentation, gender-based insights, and country-wise sales summaries.

---

## 📁 Project Overview

The Excel workbook includes:

- Raw bike sales dataset  
- Pivot Table summarizing:
  - Age Groups  
  - Customer Gender  
  - Country  
  - Order Quantity  
  - Yearly Sales
- A PivotChart visualizing sales summary by:
  - Age Group  
  - Gender  
  - Country

## 📈 Chart Details

The PivotChart (Clustered Column Chart) shows:

- Comparison of **Male (M)** vs **Female (F)** customers
- Grouped by:
  - Youth (<25)
  - Young Adults (25–34)
  - Adults (35–64)
- Different countries represented with different colors
